# Digital Clock

A simple digital clock built using HTML, CSS, and JavaScript.

Live: https://mca-programmer.github.io/3-Digital-clock/
---

## 📋 Features

- Shows current time in hours, minutes, seconds  
- Updates every second  
- Styled display for time  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript
